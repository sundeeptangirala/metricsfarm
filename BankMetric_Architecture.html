▸ Compute Subgraph → Computation Service (Python)
▸ Quality Subgraph → Quality Service (Python)
▸ Governance Subgraph → Governance Service (Java)
Key Capabilities

▸ GraphQL Shield for permission layer before resolvers
▸ Subscription support for real-time quality score updates

Auth & RBAC

Auth & RBAC Layer

STORAGE LAYER — Persistence & State
🗄️
Metadata Store SQL Server
Metric definitions, versions, ownership, approval records, audit logs, quality check results.
SQL Server — Metadata Store
Core Tables
▸ metric_definitions — key, name, tier, status, owner
▸ metric_versions — version_id, sql_logic, author, date
▸ metric_dimensions — specs, allowed_values
▸ source_views — unified view definitions
▸ approval_records — requests, actions, comments
▸ audit_log — all actions with actor + timestamp
Design
▸ Immutable version history — append only
▸ Optimistic locking for concurrent edits
▸ Partitioned audit log by month
📦
Computed Store SQL Server
Pre-computed metric results, materialized views, aggregation cache for common dimension slices.
SQL Server — Computed Metric Store
Core Tables
▸ metric_results — key, period, dims, value, version
▸ materialized_views — pre-joined dimension tables
▸ aggregation_cache — daily→monthly→quarterly rollups
Design
▸ Columnstore indexes for analytical queries
▸ Partitioned by period (monthly)
▸ Results tagged with definition_version
📝
Definition Repo Git
Version-controlled SQL definitions. Full diff history, branch-based change proposals, merge on approval.
⚡
Metric Cache Redis
Hot metric cache for frequently queried results. Session state, rate limiting for API gateway.
Source view abstraction — batch ETL / CDC ingestion
